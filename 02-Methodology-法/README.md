### 一、传播学（同理心传播思想）

**核心方法：**
- ❤️ 同理心理解对方

- 🤝 化敌为友

### 二、数理哲学（理性论证与形势分析）

**目标：** 用数学证明必然性，用毛泽东方法分析形势

**核心方法：**
- 📊 数学论证（5 大定理，合作必然）
- 📈 论持久战（三阶段分析法）
- ⚖️ 形势判断（敌我对比、时机把握）

**文档：** → [数理哲学/](./数理哲学/)

---

## 为什么需要两者？

**传播学：** 打动人心（感性）  
**数理哲学：** 说服头脑（理性）

**协同：**
- 对感性者 → 用同理心
- 对理性者 → 用数学证明
- 对两者兼有 → 双管齐下

**效果：**
- 感性认同 + 理性认同 = 深度认同
- **不可辩驳** ✨
