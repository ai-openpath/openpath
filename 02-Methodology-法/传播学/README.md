## 核心方法

### 升维应对

**完整方法论：**
→ [升维应对与教化之道](../../01-Philosophy-道/升维应对与教化之道.md)

**核心：**
1. 不与反对者同维对抗
2. 升到更高维度俯视
3. 看到对立的局限
4. 提供超越对立的解法

### 三段式回应

**模板：**
```
【第一段：肯定+理解】
承认对方合理性，建立信任

【第二段：扩展+升维】
展示更大图景，提供更优解

【第三段：邀请+共建】
从反对者变为贡献者
```
