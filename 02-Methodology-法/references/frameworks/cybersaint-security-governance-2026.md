## 推荐框架 / Recommended Frameworks

### 安全框架 / Security Frameworks
1. **NIST CSF 2.0** - 网络安全框架（美国标准）
2. **ISO 27001** - 信息安全管理体系
3. **SOC 2** - 服务组织控制（云服务/SaaS）

### AI治理框架 / AI Governance Frameworks
4. **NIST AI RMF** - AI风险管理框架
5. **EU AI Act** - 欧盟AI法案合规
6. **ISO 42001** - AI管理系统标准（2023新发布）

## 关键趋势 / Key Trends

- **AI攻击面扩大**: 模型投毒、提示注入、数据泄露
- **合规复杂化**: 需同时满足安全 + AI专项法规
- **自动化需求**: 手动合规管理难以应对AI系统规模
