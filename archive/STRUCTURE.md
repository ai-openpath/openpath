## 📅 Daily 目录组织

```
daily/
└── YYYY-MM/              # 按年月分类
    └── DD-topic-name.md  # 日期-主题
```

**命名规范：**
- 日期：两位数（01, 02, ..., 31）
- 主题：kebab-case（小写，连字符）
- 例：`15-matplotlib-incident.md`

**内容要求：**
- 📰 What：事件简述
- 🔍 Analysis：为什么重要
- 💡 Lessons：对 OpenPath 的启示
- 🔗 Sources：原始链接

## 🎯 优先级原则

**80-20 法则：**
- 80% 精力在**道法势**（理念、战略、趋势）
- 20% 精力在**术器**（方法、工具）

**为什么？**
- 理念和战略才是核心竞争力
- 工具和技术容易被复制
- **我们要做的是引领思想，而非提供工具**
