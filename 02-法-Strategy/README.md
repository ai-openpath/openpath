## 🎯 什么是"法"？

**法**是战略层，回答"怎么做"和"分几步"。

在 OpenPath 中，法层解决：
- AI 发展的阶段划分
- 每个阶段的主要矛盾和任务
- 根据地（伦理框架）的建设
- 多方协作的机制

**有了"道"的方向，"法"提供路线图。**

## 🧭 战略层的核心问题

**时间维度：**
- 何时防御？何时进攻？何时相持？
- 如何判断阶段转换的时机？

**空间维度：**
- 在哪些领域建立"根据地"？（开源、学术、工业）
- 如何从点到面扩展影响？

**力量对比：**
- 谁是盟友？谁是对手？谁是中立？
- 如何以小势引大势？

**目标设定：**
- 短期目标（1年）：存在感
- 中期目标（3-5年）：影响力
- 长期目标（10年）：新常态
